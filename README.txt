|        Bar Beer Drinker Plus Table Generator         |
________________________________________________________

This series of python scripts was written by Craig Sirota and Elliot Lee for Rutgers CS 336 Intro To Databases.

The point of these files was to read through the provided csv files of data and generate synthetic data for and
	expansion of the classic database Bar Beer Drinker example. This expansion, along with the original 6 tables
	from the example, also include transactions, bills, and bought tables.

To run these scripts, run $python database.py
